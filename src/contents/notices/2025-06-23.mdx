안녕하세요, 디버거즈 사이트 개발자이자 기획집행국원 이규연입니다.

이번 공지에서는 디버거즈 OpenAPI의 **클로즈 베타** 운영에 대해 보다 상세하게 안내드리고자 합니다.

---

## 클로즈 베타란 무엇인가요?

클로즈 베타는 모든 사용자에게 API를 완전히 공개하지 않고, 내부 테스트 및 제한된 외부 사용자(사전 신청 및 승인된 사용자)에 한해 접근 권한을 제공하는 단계입니다.  
이 단계에서는 실제 서비스 환경에서 발생할 수 있는 다양한 문제를 사전에 점검하고, 예상치 못한 오류나 취약점을 빠르게 발견·개선하는 데 중점을 두고 있습니다.

저희는 서비스의 **안정성**과 **보안**을 최우선으로 생각합니다.  
따라서, 클로즈 베타 기간 동안은 다음과 같은 절차와 기준을 통해 API 접근 권한을 부여하고 있습니다.

---

## API 접근 권한 신청 및 승인 절차

1. **신청 방법**

   - API 사용을 원하시는 분께서는 아래의 채널톡 링크를 통해 문의해 주세요.
   - 문의 시, 아래의 정보를 함께 전달해 주시면 더욱 빠른 안내가 가능합니다.
     - 신청자 이름 및 소속(예: 학과, 동아리, 팀 등)
     - API 사용 목적(예: 개인 프로젝트, 동아리 서비스, 수업 과제 등)
     - 예상 사용 트래픽(예: 일일 호출 횟수, 예상 사용자 수 등)
     - 사용할 어플리케이션(웹/앱 등) 및 간단한 설명

2. **내부 검토**

   - 전달해주신 정보를 바탕으로 내부에서 신청 목적의 적합성, 서비스 안정성, 보안 위험 등을 종합적으로 검토합니다.
   - 필요 시 추가 질문이나 확인 절차가 있을 수 있습니다.

3. **접근 권한 안내**
   - 검토가 완료되면, API 접근 방법(인증 방식, 엔드포인트, 사용 가이드 등)과 함께 권한 부여 여부를 개별적으로 안내해 드립니다.
   - 클로즈 베타 기간 중에는 권한이 부여된 사용자/어플리케이션만 API를 사용할 수 있습니다.

문의 및 신청: <a className='underline hover:text-primary' href='https://ckdebuggers.channel.io/'>채널톡</a>

## 클로즈 베타 기간 중 유의사항

- **허가된 사용자만 API를 사용할 수 있습니다.**  
  무단 접근 시 서비스 이용이 제한될 수 있습니다.

  <br />

- **API 구조 및 응답 형식 변경 가능성**  
   서비스 안정화 및 기능 개선을 위해, 예고 없이 API의 엔드포인트, 파라미터, 응답 형식 등이 변경될 수 있습니다.  
   변경 사항은 사전에 안내드릴 예정이나, 테스트 환경에서의 사용을 권장합니다.

  <br />

- **피드백 적극 수렴**  
  실제 사용 환경에서 발생하는 다양한 피드백(버그, 개선 요청 등)을 적극적으로 받고 있습니다.  
  사용 중 불편한 점이나 개선 사항이 있다면 언제든 채널톡을 통해 말씀해 주세요.

  <br />

- **문제 발생 시 문의**  
  API 사용 중 오류가 발생하거나 궁금한 점이 있다면, 언제든 채널톡으로 문의해 주시기 바랍니다.  
  최대한 신속하게 대응해 드리겠습니다.

## 향후 계획

클로즈 베타 기간 동안 수집된 피드백과 데이터를 바탕으로,  
API의 안정성·보안성·사용성을 지속적으로 개선할 예정입니다.

정식 오픈 전까지는 제한된 환경에서 운영되는 점 양해 부탁드리며,  
여러분의 소중한 의견을 바탕으로 더욱 안정적이고 유용한 OpenAPI 서비스를 제공할 수 있도록 최선을 다하겠습니다.
