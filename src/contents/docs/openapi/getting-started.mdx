## 환영합니다!

디버거즈의 개발자 포럼에 오신 것을 진심으로 환영합니다.

이 페이지에선, 지금 보고 계신 개발자 문서가 어떤 것들을 다루는지,
저희가 어떤 서비스들을 개발자 분들께 제공하고 있는지 등을 설명하며
저희의 api를 개발자 분들이 어떻게 사용하셔야 할지 또한 서술되어 있습니다.

저희 디버거즈가 제공하는 모든 서비스와 api는 전부 "청강문화산업대학교 게임스쿨" 학우븐들과
그들이 제공하는 서비스를 대상으로 하며, 외부인 분들이 api를 이용하기 위해선 [사이트 어드민](https://ckdebuggers.channel.io)에게의
문의가 필요합니다.

## 왜 디버거즈인가?

어떻게 사용하는지에 대해 알아보기 전에, 왜 저희 api를 사용해야 하는지 알아봅시다.

**신뢰할 수 있는 학생 인증**

현재 시중에 존재하는 모든 서비스를 통틀어서, "청강문화산업대학교"의, 그것도 "게임스쿨"의 학생들이
지금 어떤 것들에 관심이 있는지, 정말 우리 스쿨의 학생이 맞는지 등을 확인할 수 있는 서비스는
저희 디버거즈의 서비스가 유일합니다.

**학생회가 직접 운영하는 서비스**

저희 청강문화산업대학교 게임스쿨 학생회 "디버거즈"는 학생들의 니즈를 가장 잘 이해하고 있는 조직입니다.
학생들이 필요로 하는 기능과 서비스를 직접 개발하고 운영하며, 빠른 피드백과 개선이 가능합니다.

**다양한 API 제공**

디버거즈는 다음과 같은 다양한 API를 제공합니다:

- **학생 인증 API**: 청강대 게임스쿨 재학생 여부를 확인할 수 있습니다
- **학생 정보 API**: 학번, 이름 등 기본적인 학생 정보를 조회할 수 있습니다
- **커뮤니티 API**: 학생들의 관심사와 활동 데이터를 확인할 수 있습니다
